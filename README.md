# Java Conditionals & Loops Practice

A repository for practicing conditional logic, loops, and interactive Java applications as part of the Udemy course "Java Masterclass 2025: 130+ Hours of Expert Lessons".

## Topics Covered

- **Conditional Statements**: if, else if, else, switch, ternary operators
- **Loops**: for, while, do-while, enhanced for-each
- **Control Flow**: break, continue, return statements
- **Logical Operators**: &&, ||, !
- **Comparison Operators**: ==, !=, <, >, <=, >=

## Project Structure

```
src/
└── Main.java
```

## Running the Code

Compile and run using:
```bash
javac src/Main.java
java -cp src Main
```

Or use your IDE's run configuration.
